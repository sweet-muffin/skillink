## 📍 Introduction
Udemy 큐레이션, wanted, 조코딩이 주최한 본 대회는 AI 기술을 활용하여 웹/앱 서비스를 만드는 것을 목적으로 하였습니다.
SKILLINK는 원하는 프로젝트 목표를 알려주거나, 원하는 직무에 대한 채용 공고를 선택하면, 그에 맞는 학습 커리큘럼을 추천해주는 서비스입니다.
